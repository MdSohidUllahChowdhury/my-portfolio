Rose Leaf Disease Detection
↗ GitHub

An AI-powered mobile application enabling farmers and agricultural researchers to diagnose rose leaf diseases in real time using a camera — no internet required. The app runs a custom-trained CNN + YOLOv8 model directly on-device via TensorFlow Lite, delivering instant, accurate results in the field.

Problem

Farmers lack immediate access to agricultural experts when plant diseases strike. Early detection is critical — delayed diagnosis leads to crop loss, financial damage, and wasted resources.

My Solution

Trained a CNN + YOLOv8 model on annotated rose leaf datasets, optimized it for mobile inference using TensorFlow Lite, and embedded it into a Flutter app with a clean, accessible interface designed for non-technical users.

Tech Stack
Flutter Dart Python TensorFlow Lite YOLOv8 CNN NumPy Jupyter
Key Contributions

Designed and trained the ML pipeline end-to-end. Optimized inference latency through TFLite quantization. Built the entire Flutter application with intuitive UX for farmers.

⚡ 94% Classification Accuracy
🚀 35% Faster Inference via TFLite
📱 100% On-Device, No Internet

WallX
↗ GitHub

A high-resolution wallpaper application pulling 3,000+ HD images from a REST API — with optimized caching for near-instant load times. A focused exercise in API integration, performance engineering, and intuitive mobile navigation.

Problem

Wallpaper apps are notoriously slow. Large image payloads cause jank, wasted data, and frustrated users who abandon the experience before finding what they want.

My Solution

Integrated flutter_cache_manager to implement aggressive intelligent image caching, slashing repeat load times by 50%. Built clean navigation and routing with MVC architecture for maintainability and extensibility.

Tech Stack
Flutter Dart RESTful API flutter_cache_manager MVC
Key Contributions

Designed and implemented the full API integration layer. Engineered caching strategy that halved load times. Built an intuitive browsing and preview UI with smooth navigation.
